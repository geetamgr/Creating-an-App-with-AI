# Creating-an-App-with-AI
Writing birthday cards is harder than it should be. This app makes it quick, funny, and actually enjoyable.

🎉 Ultimate Birthday Card Generator

A fun and interactive web app that helps you create personalized, funny birthday card messages in seconds.

Just enter a few details about the person, and the app generates a ready-to-use message you can copy straight into a card.

✨ Features

🎂 Generate humorous birthday messages instantly

🧠 Personalization with:
Name
Age
Hobby
Relationship

Message style (e.g. silly, sweet, sarcastic)
🃏 Beautiful card-style preview (like the inside of a real card)
📚 Keeps previous cards stacked for easy comparison
📋 One-click copy of the latest message
🎶 Built-in “Happy Birthday” song player
🎈 Animated UI with floating balloons and stars
🎬 Smooth card pop-in animations

🚀 How to Use
Enter the person’s details
Click “Generate message”
View the card on the right
Copy your favorite message or generate more

🛠️ Tech Stack
HTML
CSS (animations + styling)
Vanilla JavaScript (no frameworks)
Web Audio API (for birthday song)

📦 Setup
No installation needed.
